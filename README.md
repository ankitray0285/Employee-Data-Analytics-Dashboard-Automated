# Employee Data Analytics Dashboard

## Project Overview
This project focuses on building an interactive Employee Data Analytics Dashboard to analyze workforce distribution, salary trends, and data quality.

The dashboard automates data cleaning, provides key business insights, and enables dynamic filtering for better decision-making.

---

## Features

### Data Cleaning
- Duplicate removal  
- Missing value identification (Email, Salary)  
- Standardized formats (Date, Salary, Department)  

### Key Metrics
- Total Records  
- Duplicate Entries Removed  
- Missing Values (Critical Fields)  
- Average Salary by Department  
- Employee Count by Department  

---

## Visualizations
- Bar Chart: Employee count by Department  
- Line Chart: Salary trends over time (Joining Year)  
- Pie Chart: Department distribution  

---

## Business Insights
- Which department has the highest average salary  
- Employees joined in the last 1 year  
- Client-wise service distribution  

---

## Interactive Filters
- Department  
- Joining Year  
- Client Company  

---

## Automation
- One-click Refresh Data:
  - Imports new data  
  - Cleans dataset  
  - Updates dashboard automatically
